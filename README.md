# TCC-Pesca_Curso
Projeto de TCC Pesca Curso
***
LISTA DE FUNCIONALIDADES DO SITE
***
- FUNCIONALIDADES PRINCIPAIS:
- SISTEMA DE FILTRO
- BARRA DE PESQUISA 
- VISUALIZAÇÃO DAS PÁGINAS DE CURSO
- CRIAÇÃO DE CONTA E LOGIN
- CRIAÇÃO DE CURSO
- CONTA DE ADMINISTRADOR (PODE DELETAR CONTAS OU POST)

***
- FUNCIONALIDADES SECUNDÁRIAS:
- AVALIAÇÃO DE CURSO
- SISTEMA DE PERSONALIZAÇÃO BÁSICA DA APARÊNCIA DO SITE (ex:DARK MODE)

***
ORGANIZAÇÃO

Para a criação do projeto ocorrer de forma mais organizada a seguinte ordem será seguida

- 1.Criação de todas as páginas (front-end)
- 2.Criação das primeiras e mais simples funcionalidades (Criação de conta e curso)
- 3.Implementar o banco de dados para armazenamento de informações da etapa anterior
- 4.Implementar as funcionalidades principais restantes
- 5.Analisar e implementar as funcionalidades secundárias 
- 6.Revisão final do projeto e pequenas mudanças no sistema se necessário
